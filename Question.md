## :fr: Ce qui est compris

Je peux comprendre que fork doit essentiellement copier le projet dans mon espace personnel. Il est ainsi possible de modifier un ou plusieurs fichiers, de les pousser, puis d'ouvrir une pull request pour que l'équipe du projet vérifie les modifications. Nous pouvons alors fusionner notre mod dans le projet global, rendant le mod actif et disponible pour tous.
